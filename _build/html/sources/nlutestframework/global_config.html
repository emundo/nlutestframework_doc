global_config¶

class nlutestframework.global_config.GlobalConfig(python, iterations, ignore_cache)[source]¶

Bases: object

Global configuration of the NLU test framework.

__init__(python, iterations, ignore_cache)[source]¶
Parameters

  • python (str) – The absolute path to a python executable. This executable can be used by NLUFramework implementations to call respective external modules. See the SnipsNLUFramework implementation for an example.

  • iterations (int) – The number of iterations to measure the performances of the frameworks.

  • ignore_cache (bool) – A boolean indicating whether to ignore cached data.
