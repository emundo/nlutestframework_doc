nlu_framework¶

class nlutestframework.nlu_framework.NLUFramework(*args, **kwargs)[source]¶

Bases: nlutestframework.has_logger.HasLogger

Frameworks follow a certain lifecycle while being benchmarked. Implementations can react to new stages of their lifecycle by overriding methods provided by this base class.

The lifecycle and the corresponding event methods look like this:

.construct(...)
    for each data set:
        .prepareDataSet(...)
            repeat n times:
                .train(...)
                # Validation
                .cleanupTraining(...)
        .unprepareDataSet(...)
.deconstruct(...)

Frameworks are not forced to react to all lifecycle events.

See List of Built-In NLU Frameworks.
